# Java Game Hub

**GameHub is a simple multi-threaded client-server program written in Java that facilitates group
chat and direct messaging**
- The program hosts several simple games, also written in Java
- The client GUI program contains Tetris, Tic-Tac-Toe and Snake
- The non-GUI server program handles client requests for multi-player gameplay where muliple users can challenge each other to Tetris and Tic-Tac-Toe matches
